Selenium Ruby is a ruby project that aim to make the popular "Selemium RC":http://www.openqa.org/selenium-rc/ available on rubyforge.

With Selenium Ruby:
* You can install selenium gem available on rubyforge, which includes the selenium server and SeleniumDriver from Selenium RC project.
* Selenium's action based API is wrapped into a object based API.  ("Object based web testing":doc/domain-based-web-testing.html)
